
# Simple URL Shortener

A lightweight Node.js application that takes long URLs and turns them into short, clickable redirects. Built to demonstrate basic routing and object-based data storage in Express.

## 🚀 Features
- Create short URL aliases automatically.
- Redirects users from short IDs to original websites.
- Minimalist web interface.
- Handles URL parameters via Express routing.
